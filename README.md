# Project-Heimdall

## Script Server
A simple way of running scripts and automations without having to remember loads of command line prompts

## AWS Cli Tools
Pre-written scripts to complete actions on AWS that are run regularly

## AWS Instance Connect
Pre-written scripts that allow easy connection via SSH to EC2 instances and RDS databases

## AWS Localstack
A way to test and mock AWS infrastructure locally

## Pulumi Multistack
A basic setup for using Pulumi in a multistack / project based setup, allowing one Pulumi installation to deploy multiple stacks using self-managed state stored in S3

## Pulumi Localstack
A basic setup to allow running Pulumi against AWS localstack in order to test stack runs and new constructs, using self-managed local state

## Datadog Log Extractor
Tools for the datadog API that will extract log entries based on search parameters, and allow the feeding of these logs to external tools and sources

## LocalAI model runner
Runs LLM models on the local machine, such as the Gemma 2 model, which can be used, for example, to process logs entries from the Datadog Log Extractor tool

## MCP Server for Local Tools
Allows AI agents to access facilities and services provided by the tools in this project
